# Unit Converter Bot

A Telegram bot that converts between various units including length, weight, temperature, volume, area, speed, time, and data size.

## Features
- 8 categories with 40+ units
- Interactive button interface
- Quick reverse conversion
- No external API required

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Create `.env` file with your bot token
3. Run: `python bot.py`

## Deployment on Railway
1. Push code to GitHub
2. Connect repository to Railway
3. Add `BOT_TOKEN` as an environment variable
4. Deploy!
